# サポートされているMarkdown機能

現在、このアプリケーションでは標準的なMarkdown記法がサポートされています。

## 1. 太字 (Bold)
テキストをアスタリスクまたはアンダースコア2つで囲みます。

**構文**:
```markdown
**太字のテキスト** または __太字のテキスト__
```
**結果**: **太字のテキスト**

## 2. 引用 (Blockquote)
行頭に `>` を付けます。

**構文**:
```markdown
> これは引用です。
> 複数行にわたることもできます。
```
**結果**:
> これは引用です。
> 複数行にわたることもできます。

## 3. 画像 (Images)
標準的な画像記法を使用します。

**構文**:
```markdown
![代替テキスト](/path/to/image.png)
```

## 4. コードブロック (Code Blocks)
標準的なコードブロックはサポートされていますが、**シンタックスハイライトは現在無効**です（プレーンテキストとして表示されます）。

**構文**:
```markdown
\`\`\`javascript
console.log('Hello');
\`\`\`
```

## 5. リスト (Lists)
箇条書き（順序なし）と番号付きリスト（順序あり）がサポートされています。

**構文**:
```markdown
- 項目 1
- 項目 2

1. 一番目
2. 二番目
```
